Your request is missing the "url" parameter.

Please specify the url parameter as below:

https://cors.kcak11.workers.dev/?url=https://www.example.com

Note: In the url, don't forget to also include the protocol i.e. http:// or https://

Other useful parameters:

  • contentType : You can specify a contentType for your response.
    e.g.: https://cors.kcak11.workers.dev/?contentType=text/plain&url=https://www.example.com
  • format=base64 : You can specify the format=base64 parameter to get a Base64 response for the requested resource.
    e.g.: https://cors.kcak11.workers.dev/?format=base64&url=https://www.example.com
    Note: When specifying the format=base64 parameter, the contentType parameter is ignored and the response Content-type will always be of type application/json
